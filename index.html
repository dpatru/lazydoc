lazydoc

Lazy documentation for jquery.

Benefits

Lazydoc makes it easy to include interactive examples right in your documentation. This allows the lazy reader to play with your work without having to download and setup anything.

To use, checkout a copy of codemirror into your root directory as well as a copy of lazydoc. Then simply import lazydoc.js in your html page and call lazydoc on the elements you want transformed. Lazydoc will automatically get the codemirror script if it's not already loaded. Here's an example of a documentation file.

lazydoc() Returns: jQuery

Lazydoc replaces each element in the jQuery object with a codemirror frame. After each codemirror frame, lazydoc inserts 2 buttons (update and reset height & width) and another iframe to hold the output.

The update button updates the output. The reset height & width button resets the height and width of the output frame as this is not always set correctly at first.
